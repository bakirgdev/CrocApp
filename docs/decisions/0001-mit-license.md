# 0001. MIT license

Status: accepted
Date: 2026-07-21

## Context

CrocApp wraps croc, which is itself MIT-licensed. Project goal is maximum OSS adoption and contribution with minimum legal friction.

## Decision

MIT license for the whole repo.

## Consequences

- Same license as croc: no compatibility questions when embedding or binding croc code.
- Anyone may fork, including commercially; accepted trade-off for an adoption-first project.
